# 🎨 Редактор открыток (Greeting Card Editor)

Приложение для создания и редактирования поздравительных открыток с возможностью добавления фоновых изображений, настройки текста и шрифтов. Разработано на Lazarus (Object Pascal).

## 📋 Возможности программы

- ✏️ **Редактор открыток** - создание новых открыток
- 🖼️ **Загрузка фоновых изображений** (BMP, JPG, PNG)
- 📝 **Работа с текстом** - ввод и форматирование
- 🔤 **Настройка шрифтов** - выбор гарнитуры, размера, цвета
- 🎯 **Предпросмотр** - мгновенный просмотр результата
- 💾 **Сохранение проектов** - возможность сохранять открытки
- 🖼️ **Галерея** - просмотр сохраненных открыток
- ⌨️ **Горячие клавиши** - быстрый доступ к функциям
